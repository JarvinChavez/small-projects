# Protein motif encoding

NSF undergraduate research — tokenize UniProt motif regions into fixed-length numeric tensors for clustering pipelines.

Related: [iCn3D integration](https://github.com/JarvinChavez/icn3d-implementation-proof-of-concept) · [color-recognition](https://github.com/JarvinChavez/color-recognition)

```bash
pip install -r requirements.txt
python encode_motifs.py
```
